# Sistema Web para Restaurante

Aplicación web desarrollada para la gestión de pedidos, productos e inventario de un restaurante. El sistema permite administrar usuarios, ventas y productos mediante una interfaz moderna y adaptable a diferentes dispositivos.

## Tecnologías utilizadas

* React + Vite
* TypeScript (.tsx)
* Tailwind CSS
* Node.js
* Express.js
* MongoDB
* JWT
* bcrypt

## Funcionalidades

* Registro e inicio de sesión de usuarios
* Contraseñas encriptadas
* Gestión de productos
* Carrito de compras
* Gestión de pedidos
* Control de inventario
* Panel administrativo
* Diseño responsive

## Instalación del proyecto

### Frontend
Desde la carpeta principal del proyecto:
npm install
### Backend
Abrir otra terminal y ejecutar:
cd backend
npm install

## Ejecución del sistema
### Terminal 1 — Frontend
Desde la carpeta principal:
npm run dev
Servidor frontend:
http://localhost:5173
### Terminal 2 — Backend
Desde la carpeta backend:
npm run dev
Servidor backend:
http://localhost:3001

## Base de datos
El sistema utiliza MongoDB como base de datos para la gestión de usuarios, productos y pedidos.

## Seguridad
La autenticación se realiza mediante JWT y las contraseñas se almacenan utilizando encriptación bcrypt.
